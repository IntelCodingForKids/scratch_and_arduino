Installation Guide for ScratchX + Genuino101

This document describes the hardware and software settings for ScratchX and Genuino101 framework.

This environment enables a quick ramp up on software programming thanks to the free visual programming language Scratch and Genuino board. This doc is suitable for elementary and secondary students.

The presentation is split into two parts:

- Software configuration

- Hardware configuration

Let’ s start!

Scratch is a graphic programming language. Scratch is also a software learning platform, which is shared by 14.5 millions of users. The projects could be animations to 3D simulations.

Useful links:

https://scratch.mit.edu/

https://en.wikipedia.org/wiki/Scratch_(programming_language)

http://scratchx.org/

ScratchX is a web platform that enables people to test experimental functionality built by developers for the visual programming language Scratch.

There are many different extensions available on ScratchX: Genuino extension, 3D RealSense camera extension, Twitter interface.

NB: ScratchX has better performances on Firefox

1. Software Configuration

Current software configuration used is:

Windows 8.1

Firefox: 45.0.1

ScratchX plugin Installation

Download the plugin compatible with your browser (Firefox recommended)https://scratch.mit.edu/info/ext_download/

Install plugin (installer located in your Downloads folder, or in your preferred downloads folder).

2. Hardware Configuration

Communication between Genuino board and ScratchX platform is managed by Firmata protocol. Firmata library needs to be installed on Genuino hardware.

To do this, you need to install Genuino driver and Genuino IDE https://www.Genuino.cc/en/Main/Software

Download the installer compatible with your OS.

Follow the installation steps. When all is done, the path is C:\Program Files (x86)\Genuino.

Next steps :

· Start Genuino IDE.

· Configure Genuino according to the device model. Select Genuino 101.



· You might be prompted for some updates. Please do them.

· Connect your Genuino101 device.

· Check the connexion port.



· Load the StandardFirmata Library on the development environment.



· Check the library’s version is the good one. You might have to repeat this step.



· Load the library on Genuino101 device.



You’re all set!

3. Verification step

To check the installation, go to the following link : http://scratchx.org/?url=http://khanning.github.io/scratch-arduino-extension/arduino_extension.js#scratch

Connect your device.

You should see a green or yellow or red status indicator in the "More Blocks" area. See the following link https://scratch.mit.edu/info/ext_download/

Green Light means READY !



ScratchX projects have a ".sbx" extension .

To load a project, click on "Open extension file" et load a .sbx file.
